Lighten your Wordpress website by optimizing source code. Enhance its usability by adding some analytics.

== Description ==

1. Lighten your Wordpress website by optimizing source code.

* Remove WP emojies code. Remove "generator" meta tag. This plugin enables you to easily remove bunches of codes that is usually present on every page of your Wordpress website but is rarely needed. Lighter and lean source code of the HTML pages generated by Wordpress generally results in a faster load speed of the website itself.

2. Enhance its usability by adding some analytics.

* Secondly, this plugin enables you to easily add some analytics, namely Google Analytics (gtag.js), Google Tag Manager (with some possible restrictions depending on how up to date is your theme) and Facebook pixel (general abilities, only your Pixel ID is needed. In case you need a more fine grained control, please use the official Wordpress integration tool provided by Facebook).

== Installation ==

1. Upload and extract `light-and-smart.zip` to the `/wp-content/plugins/` directory
1. Activate the Light and Smart plugin through the 'Plugins' menu in WordPress
1. Go to Light & Smart Setting under the Setting menu in Wordpress admin interface to select your desired options.
